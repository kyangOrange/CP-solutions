# Competitive Programming Solutions

This repository contains a selection of my independently written competitive programming solutions, primarily from the USACO (USA Computing Olympiad) training and contests.

## About

I have studied competitive programming and learned the content of USACO Gold level, focusing on algorithm design, data structures, and efficient problem solving in C++.

Through these problems I have gained experience implementing algorithms including:

- Graph algorithms (BFS, DFS, Dijkstra)
- Dynamic Programming
- Binary Search
- Greedy Algorithms
- Prefix Sums
- Disjoint Set Union (Union-Find)
- Fenwick Trees (Binary Indexed Trees)
- Segment Trees

Many of these problems require combining multiple algorithms while carefully analyzing time and memory complexity.

## Repository Structure

Solutions are organized by difficulty level and contest.

Each solution includes:

- Problem name
- Link to the original problem
- Brief description of the main algorithm
- Time complexity (when applicable)

- C++17
